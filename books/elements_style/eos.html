The Elements of Style

William Strunk Jr., E.B. White

这是一本短小精悍的英文写作风格指南，它论述遣词造句时哪些是好的写法，哪些是坏毛病。

Strunk 是百年前康奈尔大学的教授，教过几年数学和四十多年英语。他授课时用一本自编的小册子，只有几十页。1919年 White 上过这个课，四十年后他把老师的册子扩充成一本正规的书。

怀特在纽约客和纽约时报等媒体当过五十年的记者和专栏作家，1978年获普利策奖。他还写过一些儿童畅销书，比如 Stuart Little 和 Charlotte's Web. 我只看过改编的电影，很棒！

风格要素这本书很多人都读过，有些作家甚至认为每年都应该重读一遍。这是我的第二次，上次是企鹅版，有插画更有趣一些。这次是 Pearson 2000年版。

此书前两章阐述11条文法规则，以及11条写作原则；三四章讲一些易犯的错误，都很具体也很枯燥。有其他英语教授认为怀特太保守，管得太宽。我觉得还行，有规矩总比没规矩强。

我写博士论文时，自己觉得已经很谨慎，还是被导师挑出一些中文毛病。后来写英文文章，博士后导师给改了很多文法错误。夫人的硕士论文，我帮她改了中文；她的博士论文，我帮她改了英文。这种文法书，经常写作的都该翻翻。

此书最后一章讲怎么树立自己的风格，倒是不枯燥，只是有点向玄学发展。道可道，非常道。

这本书甚至跨界影响到编程语言，Brian W. Kernighan 1974年写了本 The Elements of Programming Style, 从书名到封面，全面向风格要素致敬。后来保持队形的还有

  • The Elements of C Programming Style
  • The Elements of C# Style
  • The Elements of Java Style
  • The Elements of MATLAB Style
