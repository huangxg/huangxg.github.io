简明 KGS 指南

下载安装

  1. 很多人电脑上已经有 Java，没有的到 甲骨文网站下载 Java, 点 Java Download 会自动跳转到自己电脑操作系统相应的下载页面。点 Agree and Start Free Download。
  2. 到 KGS 网站 下载软件，根据电脑操作系统选 For Windows 或 For Mac 的下载链接。那个Raw JAR File 可能是给 Linux/Unix 的。
  3. 安装软件时需要管理员权限。
  4. 如果是苹果电脑，第一次运行软件时可能打不开，需要到 System Preferences -> General 那里改一下设置。

注册设置

  1. 运行 KGS, 在启动窗口点 Play on KGS 弹出登录窗口。
  2. 填写用户名和密码，新用户密码先随便填。点 Guest 进入主界面。
  3. 点 User -> Register 菜单，弹出注册窗口。
  4. 填姓名和电邮，姓名随便，电邮须真实。选上 Hide address from other users，保护自己隐私。
  5. 点 OK，KGS 会向你的电邮发个确认信，里面有临时密码。
  6. 在 KGS 主界面点 File -> Log Out 退出，回到启动窗口。
  7. 点 Play on KGS, 填用户名和密码，点 OK 进入主界面。
  8. 如果想改密码，点 User -> Edit Personal Information。
  9. 软件设置在 File -> Set Preferences，比如可以选记住密码。

棋手级别

  • 新用户还没下棋，没级别。
  • 级位棋手中从低到高，30k -- 1k，差一级棋力一般差一子。
  • 段位棋手中从低到高，1d -- 9d。9d 大概相当于中国的业余6段。
  • 职业棋手中从低到高，1p -- 9p。他们都是实名的，在 KGS 下棋不会导致段位升降。

房间

  1. 点 Rooms -> Room List 打开房间列表。
  2. 人最多的房间在 Main -> English Game Room。
  3. 机器人在 Social -> Computer Go，新用户找机器人练级比较合适。
  4. 自己新建房间在 New Rooms。
  5. 进入某房间后，房间名会出现在菜单栏下面的标签栏。中间是对局列表，上面粗体的是新开对局，下面是进行中对局。窗口最右边是该房间棋手列表，鼠标右击可以选按级别或名字排序。

寻找对局

  1. Active Games 标签下有所有进行中对局。Open Games 下有所有新开对局，正在等待对手的。也可以按房间找进行中对局看棋，或找新开对局下棋。
  2. 对局类别有 Ranked (级别)、Free（自由）、Teaching（教学）等，Ranked 才会算分。
  3. 新手可以在 Computer Go 房间找机器人。
    • 29k IdiotBot
    • 24k BeginBot
    • 21k WeakBot50k
    • 15k LibertyBot
    • 3k ayabot001
  4. 找到合适的对局，点开后可以改某些设置：
    • Handicap: 让子，一般系统会根据双方级别自动设。
    • Komi: 贴目，中国规则7.5目，日本规则6.5，让子棋一般半目。
    • Main Time: 每方保留时间，一般 10--30 分钟，再短不太够用，再长搭不起时间。
    • Byo-Yomi Time: 读秒时间，一般30秒，再短反应不过来。
    • Byo-Yomi Periods: 读秒次数，一般五次。比如设置五次30秒，自己保留时间用完开始读秒，必须30秒走一步。超过还没走就扣除一次，扣五次就超时判负了。

对局过程

  1. 设置完毕点 OK，对方如果接受就进入对局。有时机器人比较抢手，可能别人先占到，你进去只能观战。
  2. 对局开始前跟对手打个招呼，Hi 就行。
  3. 对局室按钮
    • Undo: 悔棋。一般不要悔棋，落子无悔。除非真点错鼠标了。对方如果同意你才能悔。
    • Pass: 放弃一手，如果自己认为棋局已经结束，点 Pass；对手如果也认为结束，也点 Pass。双方连着两手 Pass 进入数子阶段。
    • Resign: 认输，形势如果相差太远，或者有急事要离开，就认了吧。
    • Close: 关闭窗口离开对局。一般下完了才能关，否则可能被认为是逃跑了。
    • 如果不小心或因为网络掉线离开了对局，可以在主界面右上角点 Resume 找到那盘棋，对方如果在线而且同意，棋局可以继续。
  4. 数子时机器人一般会自动标记死子，否则棋手要手动标记死子。相连的一块空里点一个死子就行，其它的会自动被标记上。不小心标错的话可以 Undo，再 Pass，重新来。
  5. 每块死子都标记后，点 Done。双方都点 Done 后出系统显示胜负结果。
  6. 对局结束跟对手道谢。
